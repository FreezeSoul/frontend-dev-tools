# 构建工具

React 开发的标准项目构建工具，由由FaceBook React团队出品

## [Vue CLI](https://cli.vuejs.org/zh/)

Vue.js 开发的标准项目构建工具

## [Angular CLI](https://cli.angular.io/)

Angularjs 开发的标准项目构建工具

## [Ember CLI](https://ember-cli.com/)

Emberjs 开发的标准项目构建工具
